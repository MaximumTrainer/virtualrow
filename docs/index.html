The Story

From Pioneering Platform
to Open-Source Legacy

Maximum Trainer was built by one developer with a vision: bring professional-grade indoor cycling training to everyone for free.

Origins

Built by Maxime Blais (Max)

Maximum Trainer was a pioneering free indoor cycling platform developed primarily by Maxime Blais, known as Max. At a time when apps like TrainerRoad and Zwift were charging monthly subscriptions, Maximum Trainer offered an equivalent—or often superior—feature set completely free of charge. It was one of the earliest platforms to adopt open trainer communication standards using Bluetooth Smart (BLE).

Key Innovations

Advanced Features, Zero Cost

The platform featured an advanced workout editor, ERG mode for automatic resistance control, real-time left/right power balance, native Moxy Muscle Oxygen Monitor support with live Near-Infrared Spectroscopy data, a studio mode for multiple concurrent riders, and full offline functionality. These capabilities were ahead of their time for a free application.

September 7, 2019

The Closure

On September 7, 2019, Maximum Trainer officially ceased operations and support. According to SmartBikeTrainers, the developer cited a lack of external developer help and insufficient donations to cover site hosting as the primary reasons for the shutdown. A brilliant platform went dark—but its code lived on.

The Rebirth

Open Source & Community Driven

This site marks the beginning of a new chapter for Maximum Trainer. The codebase has been preserved and open-sourced under the GPL-3.0 license. The community can now build upon this foundation, revive the platform, and ensure that one of cycling's most innovative free tools continues to serve riders around the world.

Bluetooth Smart (BLE)

One of the earliest adopters of open trainer communication standards. Connects seamlessly to BLE-enabled smart trainers for full two-way communication, resistance control, and real-time data streaming.

ERG Mode

Automatic resistance control that locks your trainer to a target power output. Whether you're climbing or sprinting, ERG mode adjusts resistance so you hit your target watt regardless of cadence.

Training

Moxy Muscle Oxygen Monitor

Native support for Moxy sensors using Near-Infrared Spectroscopy (NIRS). Live SmO₂ and tHb data display during workouts with full .FIT file integration for post-workout analysis.

Specialized Sensor

L/R Power Balance

Real-time left/right power balance feedback lets you identify and correct pedaling asymmetries. A powerful tool for improving efficiency and diagnosing muscular imbalances.

Analytics

Design, edit, and manage structured training plans with a full-featured workout editor. Supports standard interval formats including .erg and .fit for interoperability with other platforms.

🔧 Build from Source

Maximum Trainer is built with C++ and the Qt Framework. Follow these steps to compile from source:

# Clone the repository
git clone https://github.com/MaximumTrainer/virtualrow.git
cd virtualrow

# Open the project file in Qt Creator
qtcreator MaximumTrainer.pro

# Or build from the command line
qmake MaximumTrainer.pro
make -j$(nproc)

Build requirements:

Qt 5.x or Qt 6.x C++14 Compiler Qt Bluetooth Module Qt Linguist (for translations) BlueZ (Linux only)
